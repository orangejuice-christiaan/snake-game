# 3D SNAKE GAME

## Requirements

	- cmake
	- lua
	- Desktop development with c++ (Visual Studio Installer)

## Build instructions

	- Run Scripts/Setup-Windows.bat, this will clone raylib and build it from source.
	- Wait until script is finished and generated .sln files
	- Open .sln in Visual Studio
	- Run as any other program as a Dist build for best performance.
